## Topics Covered

### Encoding Techniques
- **Ordinal Encoding** (`ordinal-encoding/`) - Encoding ordinal categorical variables
- **One-Hot Encoding** (`one-hot-encoding/`) - Encoding nominal categorical variables
- **Binning and Binarization** (`binning-and-binarization/`) - Creating bins and binary features

### Scaling & Normalization
- **Standardization** (`standardization/`) - Z-score standardization (mean=0, std=1)
- **Normalization** (`normalization/`) - Min-max scaling (0-1 range)
- **Power Transformer** (`power-transformer/`) - Box-Cox and Yeo-Johnson transformations

### Advanced Feature Engineering
- **Feature Construction** (`feature-construction-and-splitting/`) - Creating and splitting features
- **Function Transformer** (`function-transformer/`) - Custom function transformations
- **Column Transformer** (`column-transformer/`) - Column-wise transformations
- **Sklearn Pipelines** (`sklearn-pipelines/`) - End-to-end ML pipelines
